# E-Commerce Management System
The Management System is a simple Node.js-based application used to manage tasks and validate data in an E-commerce project.

This project demonstrates:
- Modular programming
- Task management
- Data validation
- Node.js basics

---

# Project Structure

ManagementSystem/
│
├── app.js
├── task.js
├── validator.js
├── package.json
└── README.md

---

# Files Description

## 1. app.js
Main entry point of the application.

### Responsibilities
- Starts the application
- Connects all modules
- Executes functions

---

## 2. task.js
Handles task-related operations.

### Features
- Add task
- Update task
- Delete task
- Display task details

---

## 3. validator.js
Used for validating input data.

### Features
- Checks empty values
- Validates input fields
- Prevents invalid data

---

# Technologies Used

- JavaScript
- Node.js

---

# Prerequisites

Before running the project, install:

## 1. Install Node.js
Download and install Node.js from:

https://nodejs.org

---

# Installation Steps

## Step 1: Open Terminal

Open terminal inside the ManagementSystem folder.

---

## Step 2: Move to Project Folder

```bash
cd ManagementSystem
```

---

## Step 3: Initialize Node.js Project

```bash
npm init -y
```

This command creates:
- package.json

---

# Installing Packages

This project mainly uses Node.js built-in features.

Optional package installation:

## Install Nodemon (Optional)

```bash
npm install nodemon --save-dev
```

Purpose:
- Automatically restarts server after file changes

---

# Running the Project

## Run using Node

```bash
node app.js
```

---

## Run using Nodemon

```bash
npx nodemon app.js
```
