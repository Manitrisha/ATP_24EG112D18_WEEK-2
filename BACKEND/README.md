# Backend API Project

## Overview

This project is a backend application developed using Node.js, Express.js, and MongoDB. It is used to manage product data through APIs.

---

# Project Structure

```bash
BACKEND/
├── APIs/
├── models/
├── package.json
├── package-lock.json
├── req.http
├── server.js
```

---

# Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose

---

# Installation

```bash
npm install
```

---

# Run Project

```bash
node server.js
```
