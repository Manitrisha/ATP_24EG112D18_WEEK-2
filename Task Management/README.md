
# Task Management

The contains multiple JavaScript programs related to:
- Task management
- Employee payroll
- Banking transactions
- Student performance
- Shopping cart
- Deep copy and shallow copy
- OTP simulator
- Temperature conversion
- Spread operator usage
- User object updates

The main objective of this project is to understand JavaScript concepts and improve problem-solving skills.

